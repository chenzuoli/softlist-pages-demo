# 乐知软件园

乐知软件园是一个开源软件展示和下载平台。网站采用响应式设计,支持PC端和移动端访问。

## 主要功能

- 软件分类展示
  - 全部软件
  - 图片处理类
  - 电脑系统类 
  - Python工具类
  - 开发工具类
  - 办公软件类

- 软件详情页
  - 软件介绍
  - 主要功能特性
  - 版本信息
  - 系统要求
  - 下载链接

## 技术栈

- HTML5
- CSS3 
- JavaScript
- Bootstrap 5
- Bootstrap Icons
